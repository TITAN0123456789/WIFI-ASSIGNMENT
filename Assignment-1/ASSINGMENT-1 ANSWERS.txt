1. OSI Layer for Wi-Fi Protocol

Wi-Fi (IEEE 802.11 standards) operates in:

Layer 1 – Physical Layer
Layer 2 – Data Link Layer

Explanation:

Physical Layer → Handles radio transmission (frequency bands like 2.4 GHz, 5 GHz)
Data Link Layer (MAC sublayer) → Handles frame formatting, MAC addressing, error detection

3. BSS and ESS
BSS (Basic Service Set)
Single access point + connected devices
Identified by BSSID (MAC address)

Example:

AP → Laptop, Mobile
ESS (Extended Service Set)
Multiple APs connected together
Same SSID across network
Used in large areas (campus, offices)

4. Functionalities of Wi-Fi Access Point

A Wi-Fi Access Point performs:

Provides wireless connectivity
Broadcasts SSID
Handles authentication (WPA2/WPA3)
Assigns IP via DHCP
Routes traffic to internet
Manages connected clients
Performs encryption

5. Bridge Mode vs Repeater Mode
Feature	Bridge Mode	Repeater Mode
Purpose	Connect two networks	Extend Wi-Fi coverage
Bandwidth	Full bandwidth	Reduced bandwidth
Connection	Wired/Wireless	Wireless only
Use Case	Network expansion	Signal boosting

6. Difference between 802.11a and 802.11b
Feature	802.11a	802.11b
Frequency	5 GHz	2.4 GHz
Speed	54 Mbps	11 Mbps
Range	Shorter	Longer
Interference	Less	More
Cost	Higher	Lower

8. Difference between IEEE and WFA
IEEE	WFA (Wi-Fi Alliance)
Defines standards (802.11)	Certifies devices
Technical body	Industry organization
Focus on protocols	Focus on compatibility
